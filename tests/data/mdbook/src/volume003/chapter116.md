# 第二十八章 暂且的平和

夕阳将遥远天际渲染成橘红色，然而天空却凝聚起浓云，空气变得闷热，昏黄的大地与天际另一头隐隐响起的雷声预示着，夏季的暴雨即将到来。

倘若换在昨日，暴雨势必会让避难所里的人们更加绝望。淋雨后极易染上风寒，这个世界又没有抗生素，一旦风寒发展成发烧甚至肺炎，死亡率便奇高无比。不过对此时避难所里的人们来说，夹杂雷声的暴雨除了令人心里发憷以外，再无法带来更多负面情绪。

仅仅第一天，菲娅便展现出如同神迹般的清理效率，西街区的混沌被完全清理，北部街区的混沌也仅剩一丝在苟延残喘，人们已经可以住进靠近避难所的屋子里，不必担心混沌反扑——人们发现凡是火焰风暴扫过的地方，混沌都不会卷土重来，尽管不清楚原理，但不妨碍他们住进有屋顶的房屋。

拜此所赐，那些从废墟里救出来的幸存者得到妥善救护，他们中的大多数人只是饿到没力气走路，并不是身体受了无法站起来的伤，在补充营养，得到充分休息后，已经可以下床行走，做些为救护人员端水送食物这种小事。

“每人只能拿一个哦，不许抢别人的糖果。”

“拿到糖果的人要把晚餐分发给受伤的大人，做得好的人回来可以再拿一颗糖果。”

“是~”

“保证送到！”

得到莉提与菲娅分发下去的糖果，避难所内的孩子顿时拍着胸脯保证，如同在争抢自己的食物般，向另一边艾丝蒂尔与安娜塔西雅所在的食物分发点蜂拥而去。

避难所内的孩子不多，但却是受部分人诟病的存在，小孩在遇见灾难时既无法充当劳力，还会消耗食物，一些人并不待见他们。所以莉提便想到以糖果为报酬，让孩子们为伤者送去晚餐的工作，这样的工作不算辛苦，还能减少孩子们受人白眼的频率，让避难所里的氛围活络起来，一石多鸟。

“莉提公主大人简直是天使大人。”

“哪有精灵当天使的啊，笨蛋。”

“我听说精灵族里也有精灵天使哦。不过比起莉提公主大人，我觉得还是菲娅大人更像天使。菲娅大人不只能变出食物给大家，背后还有发光翅膀，那才是真正的天使。”

“六片翅膀是神使大人吧？莉提公主大人是天使大人，菲娅大人是神使大人，这样大家就不用争了。”

“胡说，菲娅大人就是天使！”

“莉提公主才是！”

争论声越来越大，几乎演变成要打起来的程度，如果不是维护秩序的士兵连忙把两拨小孩分开，不出一会儿就会变成群架——这些孩子的年龄从七、八岁到十四岁不等。世界树大陆里，人类的成年年龄是十五岁。

在不远处看到这一幕的莉提手指缠绕起紫色发丝，露出困扰笑容。

“把我说成天使什么的……姐姐才是天使，我不过是沾了姐姐的光，才能给他们发糖果。”

你这家伙明明就很开心。

见到莉提侧着脸娇羞的模样，菲娅忍不住在内心吐槽。也许莉提心里确实觉得“姐姐才是天使”，但莉提把话说出口，明显别有它意，她在故意说给站在附近的士兵们听。

作为证据，周围那些听见莉提所言的士兵，一同露出仿佛见到天使的神色，眼中的狂热色彩，就差扑过来献上忠诚。

对士兵们的反应十分高兴，莉提跳着一下搂住菲娅手臂，脸颊亲昵地靠在菲娅身上。周围士兵的表情顿时更加狂热，以至于菲娅觉得空气都变得粘稠起来。

如果自己不是精灵公主，而是男人，士兵们的眼神肯定会杀了自己，菲娅不禁如此想到，自己这位妹妹也太会玩弄人心了，长大后肯定能吊着好几个人的同时，还被人当成纯真少女。

看在莉提没有惹出麻烦，也没有给人增添困扰的份上，还是暂时不揭穿她好了。

经过这段时间的观察，菲娅已经肯定，莉提十分喜欢周围人对她给出“楚楚可怜”、“善良纯真”的评价——虽然本人压根不是这样。

有趣的是，就像孩子们也分为莉提派与菲娅派一样，不是所有成年人都吃莉提这一套，避难所里也有人喜欢菲娅这样兼具冷淡表情与强大实力的少女。

尤其是当菲娅展开六瓣光翼的时候，犹如神使的身姿，令无数人心生崇拜之意。更有甚者，甚至暗中将菲娅当成与神明有联系的圣子，而这部分人大都是白天跟随菲娅清理过混沌的人。在亲眼见到菲娅展现的暴力之后，莉提的纯真善良便显得无比脆弱——如果能娶到这样的精灵公主，无疑会拥有一人之下，万人之上的地位，更重要的是将获得无比强烈的安全感。

当然，更重要的是这两位公主都拥有倾国倾城、动人心魄的姿容。在外貌上，虽然精灵的平均水平已经很高，很少有丑到出奇的精灵，但菲娅和莉提的外表即便是放在精灵族里，也是顶尖中的顶尖。

菲娅忍不住一边叹气，一边用手指点了点莉提的额头。

“你这家伙，也不知道收敛一点，这些人里大部分都是有妻室的男人。”

“欸？姐姐在说什么？”

抬起玫瑰色的天真双眸，莉提不解地眨了眨眼。

对此，菲娅没有回答，而是捏住莉提小巧的鼻子，让后者发出一阵吃痛的声音，眼眶红红地抗议。

“明、明明姐姐应该开心才对，大家都喜欢我，但我只喜欢姐姐。”

“我可没让莉提这么做，而且你的话太容易让人误会了。”

菲娅一脸无奈，下意识看了眼正在排队领取食物的队伍，结果附近的人们却纷纷低下头，装成没有在看菲娅的模样。他们中大多数都是白天和菲娅一起去城市里清扫过混沌的人，出现在领餐的队伍里，不是因为饥饿（他们先前已经吃过一顿），而是为了来到队伍前面，近距离看一眼监督秩序的菲娅。等快到他们的时候，他们就会离开队伍，重新排队。

虽然艾丝蒂尔也受人尊敬，但艾丝蒂尔身上更多是领主的属性，人们熟知领主拥有的话语权与威严，相比于精灵公主的菲娅，自然少了些神秘与亲近感。

当菲娅看懂这群恶意排队者的目的的时候，已经是五分钟以后。至于菲娅为什么能看懂，则是因为一名士兵被他妻子揪着耳朵拎出队伍，走之前那名妇女还以复杂神情看了眼菲娅与莉提，令菲娅被莉提低声嘲笑了一番，自己的姐姐也没好到哪里去，都是狐狸精。

这件事让菲娅深刻认识到，受人欢迎也不见得是好事。她倒是能理解男人看见外表漂亮的女人后，忍不住想多看两眼的心情，但因此破坏家庭和谐的话，还是会让人感到困扰。

为了不让同类事情再度发生，菲娅很快带着莉提与安娜塔西雅离开食物发放点——尽管安娜塔西雅脸上有罪人印记，但安娜塔西雅的完美身材与干净利落的行事风格，仍吸引了不少人。

值得一提的是，菲娅三人离开时，另一部分人露出了失望表情，他们是想从菲娅与莉提这里分得糖果的住民。在这个世界，糖是稀有资源，只有贵族有财力享用，平时人们能获取的最甜的东西，除了蜂蜜就只有水果。所以当人们看见孩子可以靠给伤病送去食物获得糖果后，便想着自己也能靠劳动获得糖果。

对此，菲娅选择假装没看见。自己身上的糖果是出发前向教会索要的东西，目的则是给莉提丰富食谱。虽说莉提好像吃腻了糖果，但考虑到糖果的数量并没有多到可以见人就发的程度，万一之后要用到时没有会很麻烦，所以成年人还是多忍忍。

三人沿着上午清理出来的荒凉街道前行，此时已经能看见士兵陆续从破旧房屋里抢救物资的景象——钱和食物以及珠宝首饰，乃至弓、剑与魔法道具，它们虽然是欧洛城住民的私有财产，但主人已死，它们便成为领主的私有物，化为重建城市的一部分。

菲娅之所以没有把屋子里的人救出后，直接一把大火烧了整座城市清理出混沌，就是为了保全房屋里的财产。

见到菲娅靠近，几位指挥士兵搬运物资，一身贵族打扮的人小跑过来，他们全部是年轻人，其中还有两位文官模样的女性，所有人脸上都传递出善意与发自内心的尊敬。

“晚上好，菲娅大人。”

“菲娅大人，需要我随行吗？”

“不，你们做自己的事情吧，我只是散散步，重建城市就靠你们了。”

菲娅微笑着谢绝贵族的好意，贵族们却一脸惶恐。

“应该是靠菲娅公主您才对。您的神力众人可鉴，如果不是您，我肯定已经死在马车里了。”

“就是就是，像您一样既强大又温柔的公主，全大陆也找不到几位。”

死在马车里……莫非这里面有反对艾丝蒂尔统治的小贵族？

即遇见混沌埋伏后坐上马车逃跑，差点被混沌刺死的那群人。

菲娅迟疑片刻。

虽说自己救了他们，他们过来感谢自己也在情理之中，但考虑到他们的人品，他们也可能是来巴结自己的人，更不用提他们的短视和自私让人鄙视。菲娅不太想和这群小贵族打交道。

婉拒了贵族的邀请与礼品后，菲娅很快便在小贵族们谄媚的笑容里离开。

实在是让人不舒服的一群人。

“姐姐不喜欢刚才那群人？”

从菲娅面无表情的脸上读出菲娅所想，莉提小心翼翼地问道。

菲娅撇了撇嘴。

“当然不喜欢。”

“王都里有很多这样的贵族哦，像是拉法里亚伯爵、莱因哈特伯爵，都是见钱眼开的势利小人。他们虽然不讨人喜，但领地很富裕，尤其和几位王妃与大臣的关系不错。他们可以透过军政大臣和王妃向父亲大人多嘴，制造些没有由头的流言，王都里的很多事情都由他们做主。姐姐如果不能适应他们的话，之后回去了肯定会很麻烦。”

由于菲娅一副无所谓的神情，莉提只好耐心地强调。

“他们是夏因亲王那一派的人，一向和姐姐不对付。之前姐姐一直控制和他们见面的次数，他们也因为姐姐做得无可挑剔，受父亲大人信任，无法指责姐姐。这次回去，姐姐不仅要解决失去记忆和克蕾雅姐姐的事情，还要为安娜姐开脱，他们肯定会对姐姐指指点点，说不定还会拿姐姐一直推辞订婚事宜为由，给姐姐使绊。姐姐还是提前培养一些忍耐力，不能把不喜欢对方的情绪表达在脸上。忍着不适与不喜欢的人笑着交谈，也是贵族的一部分。”

“夏因亲王是？”

菲娅微微睁大双眼，这一路上只要闲下来，菲娅就会学习魔法，所以安娜塔西雅没机会向菲娅转达王宫内的事宜。对于各派系之间的势力划分，还是菲娅第一次听说。

安娜塔西雅沉声解释。

“夏因亲王是陛下的弟弟，也享有继承王位的权利。他被陛下封为北境亲王，在北边的不毛之地偏居一隅，正常情况这时应该放弃争夺王位，享受宗室的俸禄，但他却一直练兵屯粮，让人怀疑其动机。”

“也就是说，那个夏因亲王与掌握话语权的贵族往来密切，随时觊觎王位？”

“正是如此。”

“啊哈，”菲娅不在意地挥了挥手，“只是些蚊子罢了，不必理会他们，勾心斗角是多余的。”

菲娅不是没有把心中想法藏起来的能力，不这么做，只是因为没必要，对方没有那个资本让她多此一举。

“可是姐姐，他们掌握了很强的兵力，您之前不想与他们往来，就是因为知道他们拥有的话语权，如果在这种时候得罪他们——”

“莉提担心得太多了。”抢在莉提焦急的声音落下前，菲娅笑着轻摸了摸莉提的头，“放心吧，我有的是方法。”

“的确，以殿下展现的力量，独自击败月龙、杀死拥有分身能力的绝望魔将，朝中除了陛下，恐怕已无人能威胁殿下。如果再算上克蕾雅小姐的禁咒魔法，属下不认为夏因亲王能掀起风浪。不……单靠克蕾雅小姐，就足以震慑四方了。”

“我没打算把事情都抛给克蕾雅啦，嘛，这些事情就等回到王宫以后再说，现在先谈谈当下的事情，”说到这，菲娅恢复商量正事时的严肃神情，“今天安娜塔西雅遇见可疑的情况了吗？”
