# 第二十六章 庆典

太阳渐渐升起，虽然才上午九点，但柏恩城的街道已人满为患，到处是商贩的叫卖声、人们讨价还价的争论以及车轮碾过街道的声音，行人摩肩接踵，在沙漠地带的太阳直射下，街道显得拥挤而闷热。

要说的是，这是柏恩城少有的光景，放在平时，上午九点人们已经结束劳作，踏上回家的旅途。沙漠地带缺水的同时也少有云朵，火辣的太阳直直照在皮肤上，除了皮糙肉厚的兽人与其他特殊种族，没人能受得了，因此人们的工作时间往往是上午六点到九点，然后再从下午四点干到夜晚，中午有相当长的休息时间。

如今九点还人声鼎沸，原因在于克萨男爵颁布的政令。

为了庆祝名为塔纳托斯的英雄成功讨伐炎王，令地脉死而复生，柏恩城将举行为期一周的庆典，克萨男爵会自掏腰包免费发放食物与酒水，同时还有彻夜不息的演出，此外，城中所有非必要岗位都放假三天。

在这种盛大节日下，人们自然欢欣鼓舞，不用工作也可以免费吃喝，比什么都来得快乐，加上炎王困扰柏恩城内的住民已久，所有人都乐意见得炎王一命呜呼，因此就连小偷、地头蛇、杀手以及其他靠见不得光的手段赚钱的角色，也都加入庆祝活动中。

可以说，自柏恩城建立以来，还从未有如此和平而热闹的时候。

虽然这对想好好睡上一觉的菲娅不太友好。

“唔……”

窗外贵族街的呼声令菲娅深深皱眉，吵闹的街道令她很早就从睡梦中苏醒，只是对于睡眠的需求使她不肯睁开双眼，牢牢抱着松软而香甜的床铺，觉得只要不睁开眼睛，她很快就能重新睡着。

然而挣扎只是徒增烦恼，随着花车缓慢从贵族街驶过，窗外的街道已是人山人海，就算菲娅想睡着，也不可能睡着。

菲娅喉咙里发出不甘呼声，但继续下去也只是浪费时间，她只得选择醒来活动活动。她不禁庆幸，还好昨晚有先见之明，事先与夜里回来的安娜塔西雅打了招呼，告诉安娜塔西雅推迟启程时间，白天不要打扰自己，不然的话，以安娜塔西雅六点半准时起床，七点钟叫醒自己的习惯，一两点才睡觉的自己肯定会痛苦无比——脑浆好像被摇匀了一样，思维极度迟缓。

今天就先去一趟魔法师协会，把昨晚制作的魔法药卖出去，然后再找克萨男爵，安排好强盗山寨内那些遭遇悲惨的女性，至于米露身上的诅咒，由于米露只记得她的主人是一枚在地下深处的巨大水晶，不清楚具体位置，自己就顺带在魔法师协会和克萨男爵那里问问线索好了。

安排好一天行程，躺在床上的菲娅长长伸了个懒腰，嘴里也跟着发出舒适呻吟，精灵公主的身体虽然娇小，但菲娅却非常喜欢——过去身为AI时，“感觉”这种东西对菲娅只是一种语言上的描述，她无法切实感受到“舒适”与“疼痛”，现在拥有了真正的肉体，丰富而多彩的世界便给予她另一角度的美妙，当然，菲娅并不清楚这些，对她而言，只是对现在的状态十分满足。

差不多该穿衣服去吃早餐了。

心中如此想到，菲娅睁开双眼。

然后，眼角浮现米露的脸孔。

“咦？”

起初菲娅以为是自己看错，可当她扭过头去后，却发现那并不是错觉，米露确实乖巧地侧躺在她身边，那双妖异猫眼正直勾勾盯视着她的脸。

“呃。”

要是昨天，菲娅肯定会被这双眼睛吓一跳，但现在，菲娅却只感到不好意思——一觉醒来发现一位漂亮萝莉正盯着自己的脸，并且连刚才伸懒腰的姿态都被看的一清二楚，感到不好意思才是正常人的反应。

“啊，醒了。早上好，主人。”

对比菲娅的不好意思，米露只是以缺乏感情起伏的语调阐述事实。

菲娅不禁尴尬地扭动脖子，从床上坐起。如果自己没记错，米露应该和蕾妮睡在一张床上，自己则单独睡在另一张床上，米露不该跑到自己床上来才是。

“米露怎么睡在我床上？”

“早上起来后看见主人睡得很香，就忍不住一直看下去，然后觉得站着很累，就躺过来了。”

“这没什么好看的吧……”

菲娅目光瞥向米露露在被子外的肩膀，米露虽然和自己盖着同一床薄被，但好好穿着昨天莉提给她准备的白色短上衣和短裙，这让菲娅松了口气，要是米露什么也没穿就钻进自己被子，自己大概会吓得滚到床底下。

眼睛可爱地眨了眨，米露极为认真地说。

“很漂亮，怎么看都看不腻。”

“欸……”

如果米露年龄再大一些，或者不是女性，自己肯定会把这番夸奖当成肉麻的情话。

将微妙想法甩出脑海，菲娅装作大人的模样，一本正经地回答。

“米露只要长大一点，会变得比我更漂亮。蕾妮呢？房间里怎么只有米露一个人。”

说话的同时，菲娅以不那么显僵硬的动作下床，像这样和别人睡在一起，除了和克蕾雅的时候，就只有在马车里与安娜塔西雅或者莉提睡在一起，然而马车是不可抗力，米露又才刚认识不久，现在的状况，令菲娅感到不自然，仿佛自己做了什么伤天害理的事。

米露跟着掀开被子起床。菲娅的身材已经算娇小，但米露比菲娅还要矮几厘米，她坐在床沿上，一边轻轻晃动白瓷般的光洁小腿，一边看着菲娅的背影。

“她去外面了，今天外面很热闹。”

“嗯……确实很热闹。米露吃早餐了吗？”

“在等主人一起。”

“等我一起？以后不用等我也没关系，我这里没有‘主人没吃饭，奴隶就不能吃饭’的规矩，而且我也没把米露当奴隶看待。”

菲娅打算等找到米露的主人，拿到米露的所有权后，就把米露从奴隶身份解放出来。

“可我想和主人一起吃饭。”

“……”

透过房间的镜子，菲娅可以清楚看见米露那双鲜红眼眸中的认真，菲娅一时有种不知道该说什么好的感觉——米露是不是有些太喜欢自己了？

虽然喜欢自己是好事，但总感觉哪里不对。

可能是自己还不知道该用什么样的眼光去看米露吧，菲娅在心中自言自语，受昨天米露说的那句“妈妈”影响，菲娅一时间不知道该把米露当成妹妹还是……女儿。

这般结论真令人羞耻。

菲娅连带着整理黑发的动作都僵硬起来，明明自己还是十四五岁的少女，却想着把米露当成可爱女儿。

——说不心动肯定是谎言，设想一下，可爱米露左一个妈妈又一个妈妈的呼唤自己，如此景象实在让人忍不住想抱着米露的脑袋好好爱抚一番，然而这只能想想，难以付诸实际，毕竟再进一步设想，一个十四五岁的少女对着一个十二三岁的少女说，“从今天起，我就是你的妈妈了”，场面简直怪异的让人想钻进地缝，要是米露再吐槽一句“这是过家家的游戏吗？”，自己肯定会连夜逃离这颗星球。

在奇怪的道路上越走越远了。

“主人？”

见到菲娅半天没有说话，米露歪着头问道，此时米露已经把菲娅需要更换的衣服双手捧到菲娅面前，俨然一副侍奉主人更衣的小女仆的模样。

太懂事了。

以至于让人心生同情，米露肯定吃了相当多的苦，才形成这份察言观色的能力。

接过衣服，菲娅故作轻松地问。

“之前米露每天都做些什么？”

“找吃的，身体虽然不会死，但会饿。最开始很辛苦，只能找果子和别人不要的食物，不过后来有人发现我的血液可以用来炼金，情况就稍微好了一些。魔法咏唱者比普通人讲信誉，他们害怕我的眼睛，但不会因为害怕不给钱。”

“像狄克那样的人只是少数么。”

菲娅所有所思地点头，魔法咏唱者数量稀少，成长过程里也需要投入许多金币，因此很少有人愿意为了小女孩一口饭钱，败坏自己的名声。

“嗯，他的炼金术太差，快被魔女挤出柏恩城，所以不想给我钱。主人昨天晚上在做什么？”

“尝试炼金术？我想做一些下级魔法药。”

听见菲娅的话，米露脑袋两侧的羽翼一下张开，稚嫩而妖异的脸庞也浮现喜色。

“我可以帮主人！只要加一点点血，就可以提升魔法药的品质，那些炼金术师可以作证！”

“不用啦，”菲娅摸了摸米露的头，“我怎么会为了魔法药的品质让米露流血，那种事情以后再也不会有了。”

“姆。”

没能帮上菲娅的忙，米露嘴巴向下弯起，显露出失望神情。

菲娅则装作没看见，转移话题。

“话说回来，米露掌握的语言里有魔法语言吗？”

“有，不过只有人类和星晶兽的魔法语言，而且一定要拿到武器才能使用出来。”

“嚯。”

菲娅挑起眉毛，既然米露会魔法，自己就不需要从头开始教米露，而是可以像莉提一样，直接跳到击杀魔物、提升位阶的阶段。昨晚自己做了相当多的最下级魔力恢复药，哪怕米露和莉提把那些魔法药当水喝，也需要好几天才能喝完。

“那魔法的种类呢？莉提会哪些魔法？”

“『重击』、『投掷』、『兽化』和『危险感知』，只有这四种，是我在工坊里学习的魔法。”

由于米露平时穿着脏兮兮的衣服，怎么看都不像有魔力的人，所以当米露踏入炼金工坊的时候，里面的炼金术师基本不会阻止米露远远旁观他们书上的内容，而这便给予了米露学习魔法的机会。

按照米露所说，『重击』和『投掷』是她偷学的魔法，『兽化』和『危险感知』，则是她天生就会的魔法，是星晶兽与生俱来的能力。

技能组比起魔法咏唱者，更偏向战士。

还穿着单薄睡衣的菲娅思考起来，米露是战士的话，说不定可以和莉提配合的很好。接下来正好要去魔法师协会，可以带着米露去挑选合适的武器与防具，另外还要再给安娜塔西雅买一把趁手的长剑，安娜塔西雅一直使用普通制式长剑，对她这个级别的人来说，显得有些寒碜，太屈才。

至于『兽化』是什么，实在令人在意，想必这就是星晶兽被称为“兽”的理由，菲娅猜测，星晶兽族虽然有兽字，但外表和人类相似，二者唯一的区别是星晶兽本应是耳朵的地方被小型的白色羽翼取代，同时在肉体能力与魔法能力上，也都比人类出色许多。

如果说兽人偏向肉体能力，精灵偏向魔法能力，那么星晶兽便是兼具二者的优点，同时还有不输给精灵的悠长寿命，他们没能统治大陆，单单是繁殖能力奇差，还停留在卵生阶段。

“主人很好奇？”

“有一点。『兽化』是指米露会变成和魔物一样？”

“是变得像巨龙一样，有巨龙的尾巴和龙角，但没有龙鳞。”

米露耐心地为菲娅解释，而在菲娅脑补里，『兽化』就是让米露变成龙娘一类的存在——想必会很可爱，米露本就给人一种异族公主的感觉，要是长出龙角与尾巴，非人的感觉会更甚，可爱程度只增不减。

可惜的是，米露使用『兽化』后只能坚持不到一分钟，且每次用完都会脱力。虽然菲娅觉得，如果自己要求米露使用『兽化』，米露应该会变给自己看，但那样一来，米露一会儿就不能和自己一起出门，菲娅便只能忍住欣赏米露变成龙娘姿态的好奇心。

“那个……主人不换衣服吗？”

“当然要换，米露先去房间外面等我，我很快就出来。”

“不需要我帮忙？我听别人说，奴隶有帮主人穿衣服的义务。”

米露抬起头看着菲娅脸庞，不知为何，菲娅总觉得米露那双妖异竖瞳里流转着像是迫切的感情。

“我就不用了。”菲娅婉拒。

自己之所以这么长时间都没换衣服，就是因为米露还待在房间里，自己还没有养成当着其他人的面脱衣服的习惯，要知道，睡衣下那具光洁而娇小的身躯，就只穿了一条带花边的胖次。

“姆……那我去外面等主人。”

声音里有一丝失落，米露迈着小巧步子离开房间，除了在走之前，又从房门缝隙里探头看了一眼菲娅，再没有任何动作。
