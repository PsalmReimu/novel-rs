# 第十一章 魔女交汇

位于亚弗列格王国北境，克萨男爵领内规模最大，也是整个北境最繁华的城市——柏恩城，由于其特殊的地理位置，同时与五个国家接壤，所以其领内贸易格外频繁，加之克萨男爵为了保护自己的安全，把整座城市修整的如同要塞般，用高大厚实的城墙里三层外三层的将其围起来，所以吸引了几乎整个亚弗列格王国北境的人来此居住。

曾有人粗略估算，居住在柏恩城里的人应该逼近三十万，而非克萨男爵向王国汇报的五万，但因为克萨男爵不喜有人在自己的地盘调查人口，总是暗中阻挠，所以具体有多少人，谁也不清楚，人们只知道自己居住的城市格外繁华，其他事情则一概不知。

这样的城市乍一听十分美好，但对真正居住于此的人来说，如果没有钱，日子过得还不如南边冷月联盟的兽人。

这里鱼龙混杂，身负死罪的逃犯、叛乱失败的贵族、受人厌恶的亡灵系魔法咏唱者、人口贩乃至魔女集团甚至暗裔，都聚集在这座城市。因而城市的秩序仅仅维持在最低程度，如果白天不小心在街边惹恼了某人，当晚便成为尸体，也毫不意外，人们已经见怪不怪。

克萨男爵不是没想过改变现状，然而他的实力不足以让他驾驭这样一个混乱的城市，先不说暗裔和魔女，首先光是那些叛乱失败的贵族，都足够让克萨男爵头大。

这些贵族虽然落魄，但还有些家底，尤其是那些亲王、大公级别的贵族，身边护卫强到一人便可闯入克萨男爵家中，提走克萨男爵的脑袋。所以克萨男爵只能睁一只眼闭一只眼，只要大家能老老实实给他交税，卖他一个面子，其他的事情怎样都好。

此处乃纠集了无数罪恶的，只要有钱、什么都可以买到的城市。

一辆带有教会标识的鎏金马车正缓缓驶向城门，原本低头赶路的人们纷纷侧目看向马车，米尔坎迪斯神圣国的痕迹在柏恩城不算罕见，但像这样大摇大摆地进来，尤其驾驶马车的车夫还是一位女性精灵，算得上头一遭。

当然，马车里的人并不知道她们吸引了无数路人的目光，菲娅现在正询问圣树的事情。

耸入云霄的金色巨树，这幅画面对菲娅来说并不新鲜，早在闪耀时代之前，人们便可以建造同样高耸的房屋，但出现在以魔法为基础的异世界，便让菲娅好奇满满。

“这么大的圣树，生活在这里的人就看不见吗？”

面对菲娅的提问，安娜塔西雅认真点头。

“是的，只有王族与受圣树青睐的精灵，才能在这个距离下看见圣树，否则就只能在王都周围百公里，才能瞻仰圣树的光辉。”说到这，安娜塔西雅面露喜色，“殿下能在边境看见圣树，说明殿下确实是殿下，您就是您的证据又多了一份。”

“唔……可我总觉得圣树有种正在凋零的感觉。”

正如菲娅所言，她所见到的圣树虽然笔直，但从树干延伸出来的分支上，却只有稀疏的几片金色树叶，仿佛正在凋零的植物。

“凋敝……这个属下也不清楚，说来羞愧，属下无法在这个距离看见圣树。”

“圣树在我出生的时候就是这样子哦，虽然听父皇说，在很久很久以前，圣树枝繁叶茂，即使远在南大陆也能看清楚，但不知道为什么，近期叶子越来越少，树枝也缩短了很多。父皇和父皇以前的国王都下过诏，如果有人能调查清楚圣树上发生了什么，就封给他亲王之位，但几千年过去，即使是那些圣子，也都无功而返。不过总感觉如果是姐姐的话，一定可以弄清楚。”

“属下和莉提殿下的看法一样，总觉得如果是殿下，一定可以搞清楚圣树上发生了什么。我们等今晚在这里休息一夜，明天就可以踏入希利瓦尔，接下来不出两周，便可抵达王都。”

“如果我能弄清楚圣树身上发生了什么，肯定早就弄清楚了……”

从蕾妮的只言片语里可以隐约猜测到，以前的精灵公主不比自己笨。

抛开圣树的话题，菲娅背靠马车，长吁了口气。自己总算要踏入精灵领，虽然王都肯定有许多繁杂的事情在等自己，但比起嫌麻烦，心中更多的是期待。

精灵到底会不会住在树上？既然是成规模的城市，像村落一样住在树上，显然不太合适。而且圣树如此巨大，精灵该怎么爬上去？普通的『飞行』魔法存在极限高度，一般到空中两三千米便无法继续上升，难道要靠人力爬上圣树？单以体力论，恐怕第四位阶的人，也很难爬到顶。说到底，圣树究竟有多高都是一个问题，总不会延伸出大气层，到了外太空。

当菲娅在脑海里头脑风暴的时候，逐渐放缓前进速度的马车终于停下，外面传来蕾妮与卫兵的交谈。

大致内容是卫兵询问跟在马车后的队伍是怎么回事，以及是否有听说过炎王的消息。菲娅下意识掀开车帘看向窗外，发现城门前聚集着三四百名神情严肃的卫兵，城墙上也摆满了大型守城器械，黝黑的炮口让人心中一阵发憷。

无论怎么看都是严阵以待的架势，仿佛即将发生一场守城战。

“这些人是我们路上遇见的强盗，你可以对着通缉令一个个确认。”头戴青色兜帽的蕾妮跳下马车，眼神极为冷漠，“至于炎王，我们一概不知，现在可以进去了么。”

“请、请稍等！”

相比于蕾妮的冷漠，士兵的态度可谓恭敬。不论是蕾妮身后的马车，还是蕾妮的装扮与气势，都可以看出马车主人的不凡，即便他只是在做士兵的本职工作，惹恼贵族的下场也只有死路一条。

事实上在菲娅的马车旁边，还有一支排着一百来号人的队伍，平民入城是一条路，而贵族入城，则是另一条，贵族不需要排队。

“请给我们一些时间确认强盗的人数，我们会给您发放对应人数的赏金！”

“快点。”

“是！”

与其说蕾妮是等待入城的人，她更像这名士兵的上司，菲娅不禁在心中感叹。另外值得一提的是，这些士兵并没有要求菲娅一行从马车上下来，他们也没有钻进马车，确认里面有没有违禁物，只是收了一笔一银币的入城费后便再也没有过问。

而菲娅则趁着这个时间走出马车，向士兵稍微打探了下城门前为什么站着这么多士兵，士兵的回答是不久前有一群打扮怪异的人前往沙漠，说是要寻找把丛林变为沙漠的炎王，而一段时间过去，柏恩城外原本是沙漠的环境突然覆盖上绿荫，柏恩城的管理者克萨男爵认为，这可能是那群人和炎王交手后的结果，门前的士兵则是为了防止炎王的手下跑来进攻城市。

按照惯例，不论冒险者还是猎魔人，只要去讨伐炎王，之后一定会有炎王的手下展开攻城战报复，加之这次闹出的动静比以往都要大，所以肯定会有数以千计的火纹蛙跑来攻击城市，于是便有了现在严阵以待的景象。

除此之外士兵还交代，每次火纹蛙袭击城市，都会带来巨大伤亡，比如上次超过一万只火纹蛙报复性攻击城市，它们甚至突破了城市的第一、第二道防线，人们靠着第三道防线与瓮城的组合，才堪堪击退火纹蛙大军，那次至少有近千名士兵死亡。

“既然每次讨伐炎王都会招致炎王的报复，为什么克萨男爵还是让人讨伐炎王？”

面对莉提的提问，士兵展现出极为健谈的一面——不论莉提还是菲娅，外表都相当可爱，没有哪个男人会拒绝与这样的美少女多说几句，更不用提菲娅和莉提一看就是上级贵族，平时这些士兵根本不可能和这种级别的贵族千金说上话。作为证据，周围的其他士兵露出了一副想热情解答，却又不好意思开口的表情。

“怎么能不讨伐炎王！炎王的手下每年不是屠村就是袭击往来商旅，让男爵大人和我们这些领兵头疼不已，就这样，炎王还要我们每月给他送去几百号人当祭品，不想着除掉炎王是不可能的吧？那可是会吃人的魔物。虽然历来的冒险者和猎魔人都失败了，但总有会成功的人，只要炎王死了，大家就能过上好日子。”

“唉，也不知道那天出去的人能不能活着回来。”

“比起那个，我更希望他们可以顺手把火纹蛙也一起解决了啊，要是炎王真死了，沙漠底下那几十万的火纹蛙冲过来，我们肯定吃不了兜着走。”

士兵开始七嘴八舌的议论，从他们颓丧的表情可以看出，他们并不觉得炎王被人讨伐掉。

而这段时间里，清点人数的士兵也回到说话的士兵身边。

“队长，人数清点完了，一共二十五人，里面还有两个第三位阶的强盗。”

“把赏金送给这两位小姐，那两个第三位阶的带到奴隶商人那里，剩下的人送进老地方。”

老地方大概是指监牢吧，菲娅想到。

比起菲娅的猜测，莉提直接问出了“老地方是什么”，士兵的回答让菲娅表情微妙。

所谓老地方，其实是关押祭品的地方，因为炎王每月都需要几百号活人当祭品，并且还一定要口感好、年纪轻的活人，所以每到上交祭品的时候，克萨男爵都会头痛，久而久之，克萨男爵便决定让领地里那些犯罪者充当祭品，如果人数不够，就会去奴隶市场购买一些。

由于士兵回答时没有压低声音，听见自己会被当成祭品的强盗顿时脸色惨白，一些人开始哭着求他们身边的蕾妮放过他们，然而蕾妮只是一脚踩在他们头上，让他们的脸埋进地里。

虽然沦为祭品很悲惨，但想到这群强盗放火烧村，把村民当成奴隶卖掉的“事迹”，菲娅便觉得这也算是因果报应。

希望他们下辈子能做个好人，菲娅心中祈祷着。

正当士兵把钱交到安娜塔西雅手里时，城墙上的士兵突然大声呼喊。

“那群人回来了！”

士兵的呼喊瞬间吸引了所有人目光，不论城墙下的士兵还是正在排队的平民，全都回过头看向远处——道路尽头，一队身穿黑袍的人正向城门走来，在这群裹着黑袍的人身后，则是一队走路姿势显疲态，但人数却接近两百人的庞大队伍。

“队伍居然这么长！”

“没有看见伤者。”

“难道炎王真被他们讨伐了？！”

“最前面的人是一群女人，不会是魔女吧？”

“但他们身后的那群人是男人，魔女怎么会和男人在一起。”

人群中仿佛被丢下一枚炸弹，众人瞬间议论纷纷，守城的士兵立刻前去迎接，剩下的普通人则用看待英雄归来般的神情，等待着远处的队伍回来。

“切，明明是姐姐杀了那个叫炎王的大乌龟。”

“殿下，也许在您看来没有受这些人类瞻仰的必要，但我们至少得出面澄清一下，避免冒名顶替的人出现……”

接在一脸不服气的莉提后面，安娜塔西雅斟酌着语言建议。

菲娅轻轻点头。

“嗯，先等等看他们要做什么吧。”

菲娅既没有肯定也没有否定，她确实觉得不应该让人冒名顶替，但由于无法确定自己的『空间切裂』的确杀了炎王，所以还是先看看这群人的说法再做决定。

很快，两百多人的队伍来到城门前，人们纷纷勾着脑袋一一看过这群身穿猫耳修女服饰的女性，生怕遗漏任何细节，直到这群人来到门前。

似乎从人们与士兵充满希望的脸上读出他们的内心想法，为首的一位约莫二十四岁左右的黑发女性先是迟疑片刻，然后以缓慢的、低沉的语气说。

“炎王死了。”

“炎王死了？”

“炎王真的死了！”

“那些火纹蛙呢？火纹蛙怎么样了？”

“你们是怎么杀掉炎王的？”

一时间，各种各样的疑问朝为首的黑发女性飞去，人们欢呼雀跃着，为炎王的死亡奔走相告，武器与地面碰撞的声音，帽子飞到天上的景象，无不在说明众人内心的惊喜——尤其是听说自己不用当活祭的强盗，已经一把鼻涕一把泪地痛哭流涕，歌颂女性们的功德。

但比起兴奋的人群，黑发女性脸上却没有丝毫高兴，不仅如此，连带着她身后的女性也一副战败士兵的颓丧神情。

“火纹蛙也死了很多……死状凄惨，所有火纹蛙都没了脑袋。”

“居然连火纹蛙也一起解决了！你们是什么人？难道、难道是可以用禁咒魔法的高阶魔法咏唱者大人？！”

士兵眼睛瞪大到宛如牛眼，声音也几近破音。

然而黑发女性只是缓缓摇头，牙齿轻咬嘴唇，显得十分犹豫。

“我不知道是谁杀死了炎王，等我们赶到的时候，本该有炎王的地方只剩一个交叉的十字峡谷……杀死炎王和火纹蛙的人不是我们。”

“啊？”

女人的话让士兵呆愣住，远处不明真相的人们依旧在欢呼，但士兵这里却安静如水。

“那是谁杀了炎王？”士兵问出理所当然的问题。

黑发女性紧皱眉头。

“我也不知道。”

说出这些后，黑发女性才终于像是卸下重担般，长吁了口气。她开始环顾周围欢呼的人，但目光很快就停留在菲娅身上，并且顺着菲娅的身体线条，眼睛直勾勾地盯视菲娅腰间别着的短匕，嘴中吐出充满疑惑的音节。

“……希望魔女大人？”
