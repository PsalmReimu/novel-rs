# 第六十三章 热闹的过场

刚打开房门，菲娅便感到自己被屋子里的众人注视，犹如被聚光灯聚焦的感觉令她稍有压力，不过她并未将其表现在脸上，月咏亲自招待的宴席相当豪华，与其说这里是一间屋子，更像是把整层楼的隔断全部打通，进而形成的宽敞大厅。

至于氛围，这里没有贵族餐厅的上位与下位之分，而是更接近贵族聚会，大厅中间是手里端着酒杯，三三两两聚在一起攀谈的当地贵族与富商，四周墙根则要么是供人休息的沙发，要么摆满供人取用的餐具与新上料理，搭配墙上的雪白帘幕与天花板上垂下的水晶灯，绚烂氛围已经颇有希利瓦尔贵族舞会的风格，处处充斥得体。

冷夜贵族推崇希利瓦尔的生活方式，学习希利瓦尔的风格也在情理之中。

“呦，我们的公主大人来了，还是两位。”

菲娅刚进门，在大厅一角坐着的莹便立刻招呼仆人将菲娅引导到自己这，这里用长沙发围成一个类似私人空间的地方，坐在这的全部是菲娅认识的人，除了菲娅的伙伴，就只有莹与月咏。

话虽如此，因为菲娅这边的人相当多——算上菲娅自己，还有安娜塔西雅、克蕾雅、莉提、米露、爱琳丝蕾特与缇露露七个人，称得上格外豪华，加上每个人各有姿色，很自然吸引了大厅里其他贵族的目光，经常有贵族不时看向这里，对菲娅一行人心生好奇。

菲娅顺势坐在莉提与米露为自己腾出来的空位上，笑着接过莹的玩笑。

“我是公主没错，诺菈也是吗？”

“对剿灭暗裔据点的英雄不闻不问，还一觉睡到现在，非得公主大人叫才起床，这不是公主是什么。”

坐在专门座位上的莹打趣道——蛇人族盘在沙发上会很占位置，所以他们有自己特殊的座椅，方便将尾巴缠在椅子腿上。

“这里面有各种各样的原因，诺菈也不是故意睡到这个点。”

解释的同时，菲娅悄悄瞥了眼身后诺菈，她担心莹的话会让诺菈以为莹不关心自己，好在诺菈没那么矫情，不仅没生气，反而还窘迫地移开视线，肩膀不好意思地扭捏起来。

“我……我也偶尔有心情不好的时候啊。”

“你可是我们的族长，即使心情不好，也要等客人回家以后再表现出来，到时候就算把我当成妈妈搂着我哭，我也会温柔地安慰你。”

“谁会把你当成妈妈啊。”

诺菈的发言令众人哄笑起来，因为菲娅和诺菈来的太晚，众人已经吃饱，所以现在只剩下后到的菲娅与诺菈对着料理大快朵颐。对也也因此渐渐从日常回到正事上，内容大体是上午菲娅与克蕾雅带回来的人质与恶魔俘虏。

那些人质并非被魔法控制精神，而是长期受到非人折磨与极大刺激后，变得对外界麻木不仁，至于恶魔们的目的，正如加利尔对诺菈所说，是为了实验如何将混沌与这些人质融合，使其成为一种全新的、不惧混沌的生命。为了达成这个目的，他们需要大量不同种族的生命作样本，所以菲娅救出来的人质才五花八门，每个种族都有。

值得一提的是，这些情报不是莹从人质口中获得，而是在爱琳丝蕾特介入下，从恶魔俘虏口中撬出的情报，为此还死了好几个恶魔。

“我一一检查过，那些人质已经彻底废掉，像精灵这类的长寿种还可以用几十年时间慢慢恢复正常，但人类、兽人和海族的寿命短得多，先不说供养他们是一笔开销，首先就算慢慢疗养，最后也不一定能在他们活着的时候让他们回归正常生活。”

“要是能把他们送回他们的家人身边也好，偏偏执行的暗裔从一开始就没想着这些人能活下来，也就没记录这些人的名字和住址。他们中大部分人身上还有奴隶烙印，恐怕是暗裔偷懒，直接从奴隶商人手里买下各种族的人。这样一来，比起投入人力财力慢慢养他们，给予他们解脱对大家都好，我们没有多余的力气养这部分人。”

说到最后，莹语气里充满无奈，她小心扫视了眼在座的人，害怕自己的冷漠话语引起反感，但莹的话不中听是一回事，实际执行起来符合常理也是一回事，这是冷夜的事情，冷夜贫穷而弱小，连自己人都要靠从邻国买粮食救济，更别提分出精力去救其他种族、甚至到死都救不活的人，有这个余裕，不如多关心关心自己手下的子民。

倘若要对莹的做法说三道四，就自己出钱出力把这些人质包揽回去，做不到就不能指责莹，正因为理解这一点，众人才无法指责莹冷血。

如同夜之魔王般身穿黑色礼裙，将绿发绑成双马尾的爱琳丝蕾特轻轻叹息。

“精灵人质就交给我带回去吧，虽然不知道这些人能不能恢复，但有一试的必要。”

“真好啊，要是我们也像你们希利瓦尔一样富有就好了。白天要是没有会恶魔语的爱琳丝蕾特小姐帮忙，我肯定到现在都不知道那些人质身上发生过什么。”

莹青绿色的蛇类眼眸里流露出像是艳羡的东西。

一旁的克蕾雅微微前倾身体，轻声建议。

“冷夜各族以自己的种族聚居，那样的话，可以写信送到豹人族、鼠人族这样各人质所属的部族里，说不定会有人来认领？人类的话，送到与暗裔为敌的米尔坎迪斯神圣国，那些神信徒说不定会看在这是被暗裔残害的同胞的份上，收留这些人。”

“也对，处理前得先问一遍，那样万一里面有有头有脸的人物，今后闹起纠纷也有说法，谢谢克蕾雅小姐提醒。”

“我只是尽可能做自己能做的事情。”

“话说回来，混沌应该生活在世界树大陆下面，这些暗裔是从哪里得到这么多的混沌，难道他们下去过？以前在奥罗拉城还有艾丝蒂尔小姐的欧洛领时，我和殿下都遇见过混沌，总感觉混沌已经无处不在。”

安娜塔西雅神情中带着一丝凝重，早在她回到布洛瑟姆的第一天，她就向军部汇报了暗裔可能与混沌勾结的消息，然而消息并没有引起军部重视，所有人都觉得，暗裔和混沌离自己都相当远，不需要在意。

“去他们大本营问问就好了吧，克蕾雅去过那里，也许可以指路？问求知魔将或者守望者也行。”

手里拿着海苔饭团的菲娅随口问道。

刹那间，餐桌上飘过诡异的寂静，除了克蕾雅苦笑着，爱琳丝蕾特双眼闪过灵感以外，所有人看向菲娅的眼神都变得怪异。

“那是自寻死路吧……公主大人是很强没错，但暗裔的大本营里有数以十万计的恶魔，说不定还有很多第七、第八位阶的高阶恶魔，加上连神明都惧怕的绝对恶之类，说成想不开自杀也没问题……”

“重点不应该是克蕾雅小姐以前去过暗裔的大本营吗？上午克蕾雅小姐回来时，头上有像恶魔一样的犄角，我当时还以为是装饰品，没想到是真的……”

接在莹的话语之后，月咏表情夸张地张大嘴，看向克蕾雅的眼睛里甚至有一丝畏惧。

“克蕾雅只是像恶魔，不是真正的恶魔，”菲娅替克蕾雅解释，“以前克蕾雅去过暗裔的大本营，但那里的恶魔没找到属于克蕾雅的种族。”

其实只要解释克蕾雅是另一个世界来的人即可，但考虑到这个世界的神憎恶来自另一世界的人，随便说出去不仅会给克蕾雅招致麻烦，也会给听者招来杀身之祸，因此还是克制一些为好。

“这样啊……”

嗅嗅。

月咏说话时，菲娅听到身边米露发出嗅闻的声音，不仅如此，米露还越来越越界，起初只是转向菲娅抽动鼻子，如今已经半跪在沙发上，双手摁着菲娅肩膀，鼻尖凑在菲娅脖颈间吸气。

虽说米露是小孩子，做些亲昵之举也没什么，但身边另一侧的莉提正发出“咕噜噜”的不满声音，要是不制止米露，这两个人肯定会争执起来。

菲娅微微歪头，一脸好奇。

“我身上有奇怪的味道吗？”

“有。”

米露毫不犹豫地回答，表情仿佛察觉丈夫出轨的妻子，让菲娅有股没有来由的心虚。

“我应该没碰有味道的东西……”

“有猫味。”

“哈啊？”

米露的回答令菲娅神色一僵，坐在对面诺菈更是差点将喝到嘴里的汤喷出来，不断在莹的拍打下咳嗽。

米露说的猫味，肯定是诺菈紧紧抱着自己时，留在自己身上的味道，菲娅一时间绷紧全身。

“菲娅大人身上有猫的味道？”米露的声音吸引缇露露的注意，后者站起身绕到沙发后，小心捞起菲娅的乌黑发丝，尽管菲娅努力地以不会被怀疑的角度前倾身体，但依然无法逃过被缇露露嗅出味道的命运，“唔，的确有一股其他女人身上的香水味，而且还有种熟悉的感觉，不过为什么是猫——”

话音未落，缇露露便看见坐在菲娅对面的诺菈，旋即如遭雷击般愣在原地，嘴角僵硬地牵了牵。

“不亲密接触，身上的味道不会沾染的这么浓郁，菲娅大人和诺菈小姐这么晚才来，不会是……不会是……这样啊，诺菈小姐是成熟的女人，所以才……”

缇露露目光低垂下去，好像在感到怅然和失落，菲娅连忙扭过头。

“不不，不是缇露露你想的那样，是这家伙突然要死要活，我为了让她振作点才用了特殊手段，也就是揍了她一顿的程度。”

“可诺菈小姐不像挨过打，进门时看您的眼神也不是在看仇人……”

“放心啦，这两个人怎么样都不会凑在一起，姐姐就算要乱来，也肯定是安娜姐优先，缇露露就放心吧。”

出乎菲娅意料的，莉提自信满满地拍着胸脯，一副胸有成竹的样子。

只是这番话将安娜塔西雅也拉扯进来，安娜塔西雅瞬间脸红，腰也一下挺直。

“莉、莉提殿下在说什么啊！”

“姐姐可是把唯一一枚一样的戒指送给安娜姐，这已经是明示了哦，明示了。虽然应该给我才对，但给安娜姐我也勉强认可，要有淑女的余裕，嗯，余裕。”

不要用咬牙切齿的表情说这句话啊。

“哦？公主大人的魅力大到不仅从诺菈身边抢走克蕾雅，现在连诺菈也沦陷了？那公主大人要不要也看看我？我会给您做一日三餐包叠被子和洗衣服哦，只要给我一些零花钱，再让我有一个精灵王妃的名头我就满足，啊，再生一个带精灵王室血统的宝宝也不错？正好可以堵上父亲的嘴，要是他跑到火猫族闹事，我还不好处理。”

说到最后，莹已经用食指抵着下巴微微仰起头，一脸认真思考的模样。

要不是知道莹是彻头彻尾的直女，喜欢开不着调的玩笑，自己肯定会信以为真，菲娅尽可能拿出一本正经的表情，同时小心翼翼地观察克蕾雅——克蕾雅无动于衷，似乎没听见大家的话，但在这个环境下，无动于衷反而更有问题，菲娅觉得克蕾雅只是故意不说自己。

好在诺菈有关键时刻临危不惧的经验，她放下碗筷坐直身体，义正辞严地解释。

“小公主说的没错，我和菲娅什么也没有。白天是我的失态，因为一时遭遇忽略周围人感受，菲娅主动找我时我还差点对菲娅动手，今天让大家，尤其是菲娅和克蕾雅担心，十分抱歉，身为族长，我不该犯这种错误。”

格外正经的话语令愉快氛围沉重起来，众人的目光纷纷在菲娅与诺菈之间徘徊，想说什么缓和氛围，但又因为不明内情不好开口。

菲娅只好放下料理。

“我倒是没觉得有带来麻烦，毕竟那种事情谁遇见了都会难过……”

“那个，公主大人和诺菈说的事情到底是什么？诺菈白天的状态不好，里面有很深的渊源？”

“呼。”

诺菈长长吐出口气，眼神渐渐坚定，这是打算把事情和盘托出的眼神，她将身后背负的烈焰大剑取下来，靠在桌边，明明只是一个寻常到不能再寻常的动作，却让人感觉到她好像要与这把剑诀别。

“这件事迟早会让人知道，我没有隐瞒的必要，悲泣山脉出现暗裔不是偶然，其实我并非真正的——”

“等等。”

抢在诺菈说出关键话语之前，菲娅直起身打断诺菈发言。她知道诺菈想说“我并非真正的圣子，我一直以来相信的女神都是暗裔，并且这个暗裔欺骗了我”，然而目前还无法确定赫莉安说的就是事实，也可能是赫莉安在欺骗诺菈，真正的女神另有其人。

现在把话说死还太早，拥有圣子的身份不能给诺菈带来肉眼可见的效益，可失去圣子的身份肯定会动摇她的威信，尤其还是让外人知道是暗裔伪装成她的女神，这会令人们畏惧诺菈。

但菲娅不能就这样直白地向众人解释，先不说不远处还有贵族，就算酒馆里很吵，他们也有听见的可能，首先有的话该说有的话不该说就是一个问题，哪怕对象是绝对不会泄密的人也一样，关键在于对人隐私的尊重，而非保密不保密。

“公主大人想说……”

“呃……”

看穿菲娅的迟疑是为什么，诺菈摇了摇头。

“菲娅，这是我必须坦白的东西，是就是，不是就不是。”

“但还没确定到底是还是不是。”

“姐姐和诺菈像在打哑谜……”

“我先离开一会儿。”

“啊，我也和米露小姐一起。”

米露最先离开沙发，缇露露则像是看懂了氛围般跟随米露一起，而连缇露露和米露都能看出来的氛围，月咏与爱琳丝蕾特一行也不会不懂，她们很快就各自找借口离开沙发，最后又变得只剩菲娅与诺菈两个人孤零零地相对而坐。

要说的是，最后月咏临走前为了给两个人缓和氛围，特地说了句邀请菲娅晚上去她房间的玩笑，笑着夸奖菲娅的精力很美味，这让桌对面的诺菈不悦地皱起眉。
